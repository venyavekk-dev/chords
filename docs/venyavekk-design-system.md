# Venyavekk design system

Практическая спецификация снята с computed styles и CSS `venyavekk.com` 21 июля 2026 года. Значения ниже используются в интерфейсе Chord Workspace.

## Токены

```css
:root {
  --page-bg: #1c1c1c;
  --text-primary: #a8a8a8;
  --text-secondary: #7d7d7d;
  --surface-primary: #252525;

  --font-body: "Ibarra Real Nova", Georgia, "Times New Roman", serif;
  --font-weight-regular: 400;
  --font-weight-semibold: 600;
  --type-body: clamp(14px, 1.25vw, 18px);
  --leading-body: 1.5;
  --type-bio: clamp(18px, 1.55vw, 23px);
  --leading-bio: 1.45;

  --space-1: 4px;
  --space-2: 8px;
  --space-3: 12px;
  --space-4: 16px;
  --space-5: 20px;
  --space-6: 24px;
  --space-8: 32px;
  --space-12: 48px;
  --space-16: 64px;

  --radius-card: 8px;
  --radius-pill: 999px;
  --content-max-width: 1440px;
  --copy-max-width: 620px;
  --transition-fast: 180ms;
  --transition-slow: 620ms;
  --ease-default: cubic-bezier(.4, 0, .2, 1);
  --ease-expressive: cubic-bezier(.16, 1, .3, 1);
}
```

## Типографика

- Ibarra Real Nova — единственная гарнитура; normal 400, semibold 600.
- Основной текст: `clamp(14px, 1.25vw, 18px) / 1.5`.
- Вводный текст: `clamp(18px, 1.55vw, 23px) / 1.45`.
- Заголовки не образуют крупную плакатную шкалу. Иерархия строится весом, расположением и воздухом.
- Курсив используется для ролевых и контекстных подписей.

## Сетка и ритм

- Контейнер: максимум 1440 px.
- Поля страницы: 20 px до 640 px, 32 px от 640 px, 40 px от 1024 px.
- Desktop-сетка сайта: `0.5fr / 1.5fr`; основной gap 32 px, от 1024 px — 48 px.
- Смысловые группы разделяются 48 px на mobile и 64 px на desktop.
- Брейкпоинты проекта: 640 и 1024 px. Контрольные ширины: 350, 375, 390, 430 px.

## Поверхности и состояния

- Карточки и изображения: радиус 8 px, без декоративной тени.
- Чипы: поверхность `#252525`, pill-радиус, hover через opacity/тон.
- Ссылки и интерактивы светлеют на hover; стандартная длительность 180 ms.
- Focus: `1px solid currentColor`, offset 4 px.
- Sticky-элементы используют отступ 20 px и полупрозрачный фон с blur только там, где это помогает работе.
- При `prefers-reduced-motion: reduce` анимации и плавный scroll отключаются.
- Светлая тема следует системной настройке: фон `#fff`, основной текст `#333`, вторичный `#666`, поверхность `#f2f2f2`.

## Композиция

- Почти монохромная палитра, спокойная serif-типографика, плотные метаданные и большие разрывы между смысловыми группами.
- Важность задаётся не гигантским кеглем, а весом 600, положением и свободным пространством.
- Сложная рабочая область может быть плотнее остального интерфейса, но остаётся в той же системе поверхностей и ритма.

## Осознанные отклонения для Chord Workspace

- Интерактивные зоны увеличены до 44 px на touch-экранах, хотя исходные чипы сайта компактнее.
- Гриф использует более заметные линии и контраст из-за сценической читаемости.
- Root note отличается квадратной формой, размером и двойной обводкой; обычные выбранные ноты круглые.
- Функциональная палитра связей усилена для быстрого считывания: green `#73e2a7`, yellow `#ffd166`, red `#ff7b7b` в dark; более тёмные варианты используются в light.
- Горизонтальный scroll разрешён только внутри грифа, пианино и лент выбора. Документ не создаёт второй горизонтальный scroll.
